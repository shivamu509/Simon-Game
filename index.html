Press A Key to Start

Level Best : 0

Instructions

* Press any key to start the simon

* Everytime Simon will give only one signal

* Repeat Simon's signal imme- diately

* If the signal is correct simon will provide another signal

* Repeat the previous signals' by adding the new signal given by Simon's

* Continue repeating the signals of the previous sequence and increas- ing that sequence by one. Try to reach the longest sequence of 21!

* If you make an error or take more than 5 seconds to play any signal, you'll hear the "RAZZ" sound. This means the game is over.
